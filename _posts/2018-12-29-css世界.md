## 一、纯css制作三道杠和双层圆

  display: inline-block;
  width:140px;
  height: 10px;
  padding:50px 0;
  border-top:10px solid ;
  border-bottom:10px solid ;
  background-color: currentColor;
  background-clip: content-box;
}

  display: inline-block;
  width:100px;
  height: 100px;
  border:10px solid;
  padding:10px;
  border-radius: 50%;

  background-clip: content-box;
}
